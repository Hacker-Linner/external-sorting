### 外部排序

**归并排序回顾**
* 将数据分为左右两半，分别归并排序，再把两个有序数据归并
* 如何归并(这里是两路，实际用堆进行 K 路)，就是比一比大小：
  * [1, 3, 6, 7],[1, 2, 3, 5] -> 1
  * [3, 6, 7],[1, 2, 3, 5]    -> 1
  * [3, 6, 7],[2, 3, 5]       -> 2
  * [3, 6, 7],[3, 5]          -> 3
  * [6, 7], [3, 5]
  * ...
